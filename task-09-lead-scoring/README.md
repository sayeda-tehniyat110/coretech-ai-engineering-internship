# Task 09: Lead Scoring & Automated Service Routing Pipeline

## Task Objective
Designed and built an automated algorithmic lead scoring engine for CoreTech Innovation. The pipeline automatically evaluates inbound corporate leads across a 0-100 index mapping variables to absolute priority tiers and recommending specialized technical implementations dynamically.

---

## Strategic Weight Matrix Parameters

| Allocation Segment | Maximum Points | Evaluation Criteria Matrix |
| :--- | :--- | :--- |
| **Project Budget** | 40 Points | Higher thresholds maximize point distribution tiers ($15,000 max). |
| **Timeline/Urgency** | 30 Points | Immediate demands or High urgencies scale points instantly. |
| **Company Scale** | 15 Points | Medium Enterprises and Corporates score higher baseline margins. |
| **Lead Integrity** | 15 Points | Direct inbound website inquiries and client referrals receive premium weight. |

## Target Priority Tiers
* **High Priority (Score >= 75):** Immediate contact, handled directly by human project engineering managers.
* **Medium Priority (Score 45-74):** Automated introductory email funnel with proposal attachments.
* **Low Priority (Score < 45):** Standard routing for generic marketing collateral tracking.

* **Jupyter Notebook:** `coretech_lead_scoring.ipynb` (Fully compiled lead evaluation logic).
* **Database File:** `coretech_leads.csv` (50 unstructured raw sample profiles).
* **Processed Outputs:** `coretech_scored_leads.csv` (Final calculated dataset mapping scores).
