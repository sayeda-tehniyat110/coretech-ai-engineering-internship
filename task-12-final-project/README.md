## ⚙️ 3. Computational AI System Architecture
The backend application bypasses classical conditional text mapping by implementing true semantic data extraction principles:
1. **Mathematical Text Vectorization:** Transforms raw user input strings and the 30-row database context columns into numeric feature vectors using a `TfidfVectorizer` mapping matrix.
2. **Similarity Processing:** Evaluates the continuous dot product angular values of high-dimensional query tokens using **Cosine Similarity** formulas:
   $$Similarity = \frac{A \cdot B}{\|A\| \|B\|}$$
3. **Automated Triage Routing:** If the score meets or exceeds the validation barrier ($\ge 0.15$), the system identifies the maximum score column index, extracts the verified source facts, and renders an output block. Queries falling below the barrier automatically route to fallback support communication records.
